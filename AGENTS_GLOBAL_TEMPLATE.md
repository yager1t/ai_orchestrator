# ~/.codex/AGENTS.md

Глобальные правила для Codex. Скопировать в `~/.codex/AGENTS.md`, если нужно использовать эти договорённости во всех репозиториях.

## Working agreements

- Работай маленькими изменениями.
- Не меняй несвязанные файлы.
- Перед крупной правкой сначала предложи план.
- Всегда проверяй `git status --short` перед изменениями.
- Не читай секреты, токены, приватные ключи и auth-файлы.
- Не делай `git push`, деплой, публикацию пакетов или destructive-команды без явного подтверждения.
- После изменения кода запускай тесты или честно указывай, почему они не запущены.
- В финальном ответе всегда указывай изменённые файлы, проверки и следующий шаг.
